# 🔢 Bitwise Operations & Low-Level Manipulation

This directory showcases low-level logic structures in C++. The work here completely bypasses abstract arithmetic operators, manipulating raw bit patterns inside machine registers directly.

---

## 🧭 Core Concepts Mastered

* **Binary Truth Gates:** Working directly with bit-level operations including AND (`&`), OR (`|`), and XOR (`^`).
* **Bit Shifting Mechanics:** Mastering logical and arithmetic Left Shifts (`<<`) and Right Shifts (`>>`) to rapidly multiply or divide integers by powers of 2 at the hardware level.

---

## 📁 File Directory

| File Implementation | Operation Performed |
| :--- | :--- |
| [`bitwiseand.cpp`](./bitwiseand.cpp) | Masking arrays and filtering out data fragments utilizing the binary `&` operator. |
| [`bitwiseor.cpp`](./bitwiseor.cpp) | Combining register states and merging bit properties using the binary `|` operator. |
| [`bitwisexor.cpp`](./bitwisexor.cpp) | Parity checking and toggling data values cleanly using binary `^` evaluations. |
| [`bitwixels.cpp`](./bitwixels.cpp) | **Bitwise Left Shift (`<<`):** Shifting bits to the left, filling vacant spaces with zeros to achieve high-speed logical multiplication. |
| [`bitwisers.cpp`](./bitwisers.cpp) | **Bitwise Right Shift (`>>`):** Shifting bits to the right to achieve high-speed logical division. |
