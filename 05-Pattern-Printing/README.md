# 📐 Nested Coordinate Matrix Layouts & Pattern Printing

This directory contains programs that generate geometric shapes, numeric grids, and text matrices. It demonstrates mastery over nested multi-dimensional loops, dynamic boundary shifts, and 2D spatial coordinates in C++.

---

## 🧭 Core Concepts Mastered

* **2D Cartesian Grid Mapping:** Treating rows ($i$) and columns ($j$) as coordinate sets to selectively output characters or values.
* **Symmetry & Inverse Calculations:** Mirroring loop steps and adjusting empty space padding to create complex shapes (Diamonds, Kites, Butterflies).
* **ASCII Character Arithmetic:** Incrementing character values dynamically (e.g., `'A' + i + j`) to tilt, shift, and manipulate alphabetic matrices.

---

## 📁 File Directory

### 🌟 Geometric & Symmetrical Outlines
* [`starsquare.cpp`](./starsquare.cpp) / [`starrectangle.cpp`](./starrectangle.cpp) — Solid standard multi-dimensional grid arrays.
* [`starrow.cpp`](./starrow.cpp) / [`starcolumn.cpp`](./starcolumn.cpp) — Bounded single-line coordinate prints.
* [`triangle.cpp`](./triangle.cpp) — Standard left-aligned linear growing triangle matrix.
* [`pyramid.cpp`](./pyramid.cpp) — Centered structural triangle layouts applying empty space tracking offsets.
* [`butterfly.cpp`](./butterfly.cpp) — Advanced inverse mirror tracking using twin loops and dynamic empty space counters.
* [`kite.cpp`](./kite.cpp) / [`hollowdiamond.cpp`](./hollowdiamond.cpp) — Advanced coordinate isolation displaying outer borders while stripping inner elements.

### 🔢 Matrix & Numeric Triangles
* [`numbersquare.cpp`](./numbersquare.cpp) / [`numprinting.cpp`](./numprinting.cpp) — Uniform multi-dimensional numeric coordinate maps.
* [`numbertriangle.cpp`](./numbertriangle.cpp) / [`trianglenum.cpp`](./trianglenum.cpp) — Right-angle numerical grids tracking vertical or horizontal values.
* [`trinum.cpp`](./trinum.cpp) / [`numvrs1.cpp`](./numvrs1.cpp) — Continuous tracking loops running custom numerical matrix increments.

### 🔤 Character Transformation Grids
* [`charsquare.cpp`](./charsquare.cpp) — Uniform character grids tracking stable output streams.
* **`charvrs1.cpp` through `charvrs8.cpp`:** An advanced mathematical progression of character matrices. These files leverage explicit data type math to reverse, step-shift, and invert alphabetical structures across changing multi-loop boundaries.
