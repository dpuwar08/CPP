## 💡 Key Engineering Takeaways

Through compiling these foundational programs, I have solidified:
1. **Memory Tracking:** Working with dynamic lengths via Standard Template Library Vectors vs static arrays.
2. **Algorithmic Logic:** Deep manipulation of nested control loops to construct symmetrical coordinate shapes.
3. **Modularization:** Transitioning monolithic files into separate, reusable functional logic layers.

## 🛠️ Compilation & Local Run Instructions

To compile and run any of these files on your local machine using a terminal/GCC environment, execute the following syntax:

```bash
# 1. Clone this portfolio repository
git clone [https://github.com/dpuwar08/CPP-Fundamentals.git](https://github.com/dpuwar08/CPP-Fundamentals.git)

# 2. Step into the target folder directory
cd CPP-Fundamentals/01-Basic-IO-DataTypes

# 3. Compile your target file using g++
g++ helloworld.cpp -o program

# 4. Run the compiled executable binary
./program
