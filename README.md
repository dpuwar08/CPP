# Core C++ Programming Journey 🚀

Welcome to my C++ repository! This repository serves as a comprehensive archive of my foundational journey through C++. It features **88 programs** covering core logic, loops, conditional patterns, function mechanics, and standard library components—focusing entirely on mastering language syntax and logic.

----

## 📂 Repository Structure & Topics Covered

Here is how the practice files are categorized based on the concepts implemented:

### 1. Basic Input/Output & Data Types
* Core basics and syntax: `helloworld.cpp`, `input.cpp`, `bool.cpp`, `typecasting.cpp`, `ASCII.cpp`
* Basic calculations: `areaofrectangle.cpp`, `sum.cpp`, `swap.cpp`

### 2. Conditionals & Logic Gates
* Number checking: `oddeven.cpp`, `oddprint.cpp`, `oddsum.cpp`, `divisable.cpp`, `sumofnumberdivisableby3.cpp`
* Decision making: `eligibility.cpp`, `grades.cpp`, `largeone.cpp`, `largeamongthree.cpp`, `ternary.cpp`
* Mathematical logic: `logicgates.cpp`

### 3. Bitwise Operations
* Implementation of low-level operators: `bitwiseand.cpp`, `bitwisels.cpp`, `bitwiseor.cpp`, `bitwisers.cpp`, `bitwisexor.cpp`

### 4. Loops & Iterations
* Loop variations: `dowhile.cpp`, `sumwithwhile.cpp`, `printingwithwhile.cpp`, `infiniteloop.cpp`, `infiniteloopwithfor.cpp`, `skip.cpp`
* Numerical tables and series: `multitable.cpp`, `numprinting.cpp`, `numvrs1.cpp`, `numbersquare.cpp`

### 5. Advanced Logic & Mathematics
* Prime numbers and factorials: `prime.cpp`, `factorials.cpp`, `powerof2.cpp`
* Series & Math puzzles: `floyd.cpp`, `number.cpp`

### 6. Pattern Printing 📐
A heavy focus on building nested loop logic through classic geometric shape outputs:
* Stars and Shapes: `starsquare.cpp`, `starrectangle.cpp`, `starcolumn.cpp`, `starrow.cpp`
* Advanced Patterns: `butterfly.cpp`, `kite.cpp`, `hollowdiamond.cpp`, `pyramid.cpp`
* Number/Character Triangles: `triangle.cpp`, `trianglenum.cpp`, `numbertriangle.cpp`, `trinum.cpp`, `character.cpp`, `charsquare.cpp`, and the `charvrs1` to `charvrs8` series.

### 7. Arrays & Vectors
* Fixed memory storage: Sequential array practices from `array1.cpp` through `array10.cpp`
* Dynamic arrays (STL): `vector1.cpp`, `vector2.cpp`, `vector3.cpp`

### 8. Functions (Modular Programming)
* Code reusability & behavior tracking: Programs from `fun1.cpp` through `fun8.cpp`
