# 📥 Basic Input/Output, Data Types & Core Calculations

This directory contains the foundational C++ programs tracking my initial transition into systems programming. The focus here is establishing absolute command over terminal streams, core primitive data types, type conversion behaviors, and foundational arithmetic logic.

---

## 🧭 Core Concepts Mastered

* **Stream Interactivity:** Mastering data routing through standard input (`std::cin`) and output (`std::cout`) operations.
* **Primitive Processing:** Managing basic data allocations across memory types like integers (`int`), floating-points (`float`), and text arrays (`char`).
* **Low-Level Typecasting:** Explicitly translating floating points and character streams using explicit conversion mechanics.
* **Character Realignment:** Mapping user characters straight to their numerical memory representations via ASCII boundaries.
